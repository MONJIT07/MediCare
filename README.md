# 🏥 MediCare+ | AI-Powered Hospital Management System

MediCare+ is a premium, full-stack hospital management platform designed to bridge the gap between patients and healthcare providers. It combines a stunning, responsive frontend, a robust Node.js backend, and an intelligent AI assistant to streamline healthcare interactions.

🚀 **Live Demo:** https://medicare-plus-one.vercel.app

---

## ✨ Key Features

### 🌐 Patient Portal (Frontend)

* **Premium UI:** Modern, high-performance design with smooth animations.
* **Dynamic Content:** Real-time doctor listings and department data fetched from APIs.
* **Appointment Booking:** Intelligent booking form with real-time slot selection.
* **AI Chatbot Assistant:** Guides users through booking, answers queries, and improves user experience.
* **WhatsApp Integration:** Instant connection for emergency and general queries.
* **Responsive Design:** Optimized for Mobile, Tablet, Desktop, and 4K displays.

---

### 🤖 AI Chatbot Assistant

* Provides real-time guidance for appointment booking
* Answers common healthcare-related queries instantly
* Enhances user experience and reduces manual support load
* Acts as a smart interface between patients and the system

---

### 🛡️ Admin Dashboard

* **Secure Access:** JWT-based authentication for administrators.
* **Live Stats:** Real-time overview of appointments, doctors, and patient growth.
* **Appointment Management:** Review, confirm, or cancel requests with a single click.
* **Medical Resource Management:** Add/Edit/Delete Doctors and Departments dynamically.
* **Message Center:** Unified inbox for patient inquiries.

---

### ⚙️ Backend & Infrastructure

* **RESTful API:** Built with Node.js and Express.js.
* **Persistent Database:** MongoDB Atlas (Cloud) with Mongoose ODM.
* **Secure Authentication:** JWT-based role-protected routes.
* **Email Notifications:** Automated confirmation/cancellation emails via Gmail SMTP.
* **Serverless Deployment:** Optimized for Vercel with dynamic DB connection handling.
* **Scalable Architecture:** Designed to handle real-world healthcare workflows.

---

## 🛠️ Technology Stack

* **Frontend:** HTML5, CSS3 (Custom Design System), JavaScript (ES6+)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Authentication:** JWT (JSON Web Tokens)
* **Communication:** Nodemailer (Gmail SMTP), WhatsApp API
* **Tools & Libraries:** Mongoose, Font Awesome, Google Maps API
* **Deployment:** Vercel (Serverless)
